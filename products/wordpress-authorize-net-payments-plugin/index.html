Description

The WordPress Authorize.Net Payments Plugin is the only dedicated standalone WordPress plugin available that allows you to accept payments through Authorize.Net on your WordPress site.

In a few simple steps you can start accepting credit card payments, donations, recurring payments and subscriptions. Since WordPress Authorize.Net Payments is a standalone solution, no complex shopping cart, form builder or membership site plugin is needed (i.e WooCommerce, etc).

Features

  • Integration between your WordPress site and Authorize.Net
  • Unlimited payment/donation forms
  • Unlimited transactions
  • Plug-and-Play. No other plugins required
  • Easy to install and configure
  • Easily switch between accepting Payments or Donations
  • Live/Test mode toggle
  • Interactive credit card input [See it in action]
  • Automatic payment/donation amount formatting [See it in action]
  • Calendar date selection for Subscription start and end dates [See it in action]
  • Optionally collect customer billing & addresses
  • Specify payment success & failure messages
  • Specify payment comments description
  • Mobile and tablet responsive
  • Easily customizable form
  • Custom select box option
  • Custom fields to capture additional data
  • Subscription and recurring payments support
  • Does not store full credit card data

Compatibility

Compatible with WordPress Version 4.8.0 and above.
