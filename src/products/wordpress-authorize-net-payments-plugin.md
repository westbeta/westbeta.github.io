### Description

In a few simple steps you can start accepting credit card payments, donations and subscriptions with Authorize.Net on your WordPress site. WordPress Authorize.Net Payments is a standalone WordPress plugin. No complex shopping cart, form builder or membership site plugin needed.

That’s it. No other plugins required.

### Features

- Unlimited payment/donation forms and transactions
- Plug-and-Play. No other plugins required
- Easy to install and configure
- Easily switch between accepting Payments or Donatations
- Live/Test mode toggle
- Interactive credit card input <a href="#" data-featherlight="/assets/img/products/wp-authnet/card.gif">[See it in action]</a>
- Automatic payment/donation amount formatting <a href="#" data-featherlight="/assets/img/products/wp-authnet/amount.gif">[See it in action]</a>
- Auto-formatting for Subscription Dates <a href="#" data-featherlight="/assets/img/products/wp-authnet/dates.gif">[See it in action]</a>
- Optionally collect customer billing & addresses
- Specify payment success & failure messages
- Specify comments description
- Mobile and tablet responsive
- Easily customizable form


- Custom select box option
- Custom fields to capture additional data
- Subscription support
- Subscription start and end date selection
- Priority email support with a 24-hour response time during business days


## Compatibility

Compatible with WordPress Version 4.8.0 and above.
